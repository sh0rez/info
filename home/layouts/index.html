Anwendung: Klassenfahrt Zimmerverteilung

PHP Anwendung zur Planung der Zimmerverteilung einer Klassenfahrt. Speicherung der Daten erfolgt über eine MySQL Datenbank, die Oberfläche wurde mit PHP Server Side Rendering und Bootstrap 4 erstellt.

Funktionsumfang:

  • Zimmerplaner: Mit dem Zimmerplaner können sich Schüler Zimmern zuweisen. Bereits belege Plätze können nicht mehr bearbeitet werden.
  • Raumeditor: Der Raumeditor in den Einstellungen erlaubt die Räume, die vor Ort vorhanden sind einzurichten (Größe, Geschlecht), sodass Schüler sich dort anschließend eintragen können
  • Einstellungen: Neben dem Raumplaner bieten die Einstellungen die Option, alle Daten zu löschen sodass eine weitere Fahrt geplant werden kann. Die Einstellungen sind Kennwortgeschützt (Passwort: admin:admin)

Wordpress: Gaesdonck – Das Internat am Niederrhein

Reduzierter Nachbau der Gaesdoncker Homepage mit Wordpress.

Verwendete Plugins:

  • WPFastestCache: Um PHP-bedingte Performanceeinbuße einzudämmen werden alle individuellen Seiten im Vorfeld dynamisch in HTML vorgeneriert. Dies Verbessert die Seitengeschwindigkeit erheblich.
  • iCal: Dieses Plugin erlaubt den auf https://gaesdonck.de/events verfügbaren Kalender dynamisch auf dieser Seite einzubetten. Dazu wird dieser im .ics Format geladen und anschließend angezeigt.
  • NinjaForms: Formulare direkt in Wordpress! Hiermit wurde zum Beispiel das Bewerbungsformular erstellt.

Homepage

Persönliche Portfolioseite, erstellt mit modernen Webtechnologien. Als Framework kam ReactJS zum Einsatz, gepaart mit GatsbyJS zur leichteren Anwendung.

ES6, JSX und CSS-in-JS erlauben eine hochperformante, abstrahierte und klare Codebasis.
